- Repo scaffold, requirements, API/CLI stubs, catalog draft

- LocalRasterAdapter (DEM + Landcover)
- Reducers: mean/std/majority

- Temporal reducer: nearest_month/monthly_mean
- Caching + small perf test

## Sprint 3
- QC checks + provenance columns
- Parquet writer + QC summary
- Midterm demo

## Sprint 4
- Hardening; docs; add one more predictor if time
